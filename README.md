# parcel-react

## Install
```bash
git clone git@github.com:mthpvg/parcel-react.git
cd parcel-react
npm install
```

## Development
```bash
npm start
```
Visit: http://localhost:1234

## Production

### Build
```bash
npm run build
```
Output is in the `dist` folder.

### Server locally
```bash
npm run serve
```
Visit: http://localhost:5000
